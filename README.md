# Meshtastic-MD
**Что такое Meshtastic**

Meshtastic — это система связи, которая позволяет обмениваться короткими сообщениями без мобильного интернета и без Wi-Fi.
Она работает по радиоканалу LoRa, который рассчитан на дальнюю связь при небольшой скорости передачи данных.
Сообщения передаются между устройствами напрямую или через промежуточные узлы.
Это подходит для случаев, когда связь нестабильна, недоступна или нужна автономная сеть.
Сеть Meshtastic можно постепенно расширять: чем больше узлов, тем лучше покрытие.

**Что такое «нода» и как работает сеть**

Нода (узел) — это небольшое устройство с радиомодулем LoRa и прошивкой Meshtastic.
Каждая нода может принимать и передавать сообщения.
Если две ноды далеко друг от друга, сообщение может пройти через другие ноды “по цепочке”.
Так формируется mesh-сеть: узлы помогают передавать сообщения дальше, увеличивая общую зону связи.

**Для кого этот гайд и что он покрывает**

Этот гайд предназначен для людей в Молдове, которые хотят подключиться к общей сети Meshtastic и помочь расширить её покрытие, даже без технического опыта.
Документ объясняет базовые понятия простым языком и помогает выбрать устройство под разные сценарии: домашняя нода, стационарный узел, узел-ретранслятор, автономная нода с солнечным питанием.
Также описываются основы совместимости, питания, прошивки, первичной настройки и типовых проблем, чтобы снизить порог входа и упростить подключение новых участников.
